### 🏗️ Архитектурные решения
- **Маска номера ВУ** (`formatLicenseNumber`): принимает только цифры + русские буквы, автоматически расставляет пробелы по формату `NNNN ЛЛ NNNNNN`. Буквы конвертируются в ВЕРХНИЙ регистр через `toUpperCase()`. maxLength=15 (4+1+2+1+6+1 с запасом).
- **Маска даты** (`formatValidUntilDate`): принимает только цифры, расставляет разделитель ` / ` автоматически после ввода месяца. Мягкая валидация: если первая цифра >1, дополняется до `0X`; месяц ограничен 01-12. maxLength=9 (`MM / YYYY`).
- **Backspace**: обе маски корректно обрабатывают удаление символов через пересборку строки из cleaned input.
- **Без изменения падежей**: согласно инструкциям, падежи НЕ тронуты (Issue #126 зона).

### 🚧 Риски / follow-up
- **Валидация года**: сейчас нет проверки на адекватность года (например, 1950 или 9999 пройдут). Можно добавить позже, если потребуется (например, год >= текущий год).
- **Серверная валидация**: клиентская маска не заменяет серверную проверку формата ВУ при отправке формы.
